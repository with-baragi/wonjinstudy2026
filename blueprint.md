
# Blueprint: 오늘 꼭 해야 할 일 추천기

## 개요

사용자의 현재 상태와 기분을 바탕으로, 미루고 싶지만 꼭 해야 하는 생산적인 활동(운동, 공부, 독서 등) 중 오늘 가장 우선순위로 해야 할 일을 추천해주는 웹 애플리케이션입니다. 선택 장애를 해결하고 행동으로 옮길 수 있는 동기를 부여합니다.

## 스타일, 디자인 및 기능

*   **에너제틱한 디자인:** 행동력을 자극하는 밝고 역동적인 색상 팔레트를 사용합니다.
*   **반응형 레이아웃:** 모바일에서 간편하게 확인하고 실천할 수 있도록 최적화된 UX를 제공합니다.
*   **다크/라이트 모드:** 사용자의 환경에 맞춘 시각적 옵션을 제공합니다.
*   **심리 기반 추천:** 사용자의 현재 기분이나 에너지 레벨을 입력받아 그에 적합한 활동을 추천합니다.
*   **동기 부여 멘트:** 추천된 할 일과 함께 실천을 독려하는 짧은 응원 메시지를 표시합니다.

---

## 현재 요청 계획: 할 일 추천기 전환 및 배포

1.  **HTML 구조 변경 (`index.html`):**
    *   타이틀을 "오늘 꼭 해야 할 일 추천기"로 변경합니다.
    *   질문 단계를 새 테마에 맞춰 수정합니다:
        1.  시작 화면: "오늘도 미루고 싶나요? 지금 바로 확인하세요."
        2.  질문 1: 현재 에너지 레벨 (1~100)
        3.  질문 2: 가장 피하고 싶은 활동은? (텍스트 입력)
        4.  질문 3: 지금 몇 시인가요? (아침/오후/저녁/밤 선택)
        5.  최종 추천: 3가지 추천 활동과 응원 멘트 표시

2.  **스타일 업데이트 (`style.css`):**
    *   메인 테마 색상을 좀 더 활기찬 오렌지 또는 그린 계열로 변경합니다.
    *   추천 결과 카드가 더 강조되도록 스타일을 개선합니다.

3.  **로직 구현 (`main.js`):**
    *   추천 대상 목록 정의: 운동, 헬스, 독서, 일, 공부, 명상, 청소, 산책, 외국어 공부 등.
    *   사용자 입력 시드를 기반으로 랜덤하게 3가지 활동을 추출하는 로직을 작성합니다.
    *   각 활동별로 적절한 아이콘이나 이모지를 매칭합니다.
    *   "다시하기" 버튼으로 초기화 기능을 유지합니다.

4.  **배포:**
    *   Firebase MCP를 사용하여 Firebase 호스팅에 배포하거나, 환경에 맞는 최선의 방법으로 배포를 완료합니다.
